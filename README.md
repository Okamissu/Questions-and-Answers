# Questions-and-Answers

Course project for **Advanced Programming Techniques 2** (UJ, Electronic Information Processing)

---

## Live Demo on Wierzba Server

For development purposes, the front-end and back-end remain as separate applications.  
The combined version of the application is available here:  
[https://wierzba.wzks.uj.edu.pl/~20_kobylarz/qa-app/](https://wierzba.wzks.uj.edu.pl/~20_kobylarz/qa-app/)

The `wierzba-setup` branch of the repository contains files adapted for the Wierzba server (WZIKS UJ).

### Files to adjust when deploying to another server

**Front-end:**

- `App.jsx` → `basename`
- `api.js` → `baseURL`

**Back-end:**

- `.htaccess`
- `index.php`

> After adjusting these files, run:

```bash
npm run build
````

and copy the contents of the `dist` folder into the `public` directory in the backend.

---

## Local Setup

### 1. Backend

Navigate to the `backend` directory:

```bash
docker compose up -d   # or docker-compose, depending on your version
docker compose exec php bash
cd app
composer install
composer init-app
```

* By default, the backend runs on **port 8000** in the included Docker configuration.

### 2. Frontend

Navigate to the `frontend` directory:

```bash
npm install
npm run dev
```

* The frontend runs on **port 5173** with the default Vite configuration.
